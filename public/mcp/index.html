Model Context Protocol (MCP) Testing Tools

About MCP

The Model Context Protocol (MCP) provides a standardized approach to provide context to LLMs (Large Language Models), enabling integration with n8n automations and other AI applications.

Think of it as a "USB-C port for AI applications" - a common interface to connect AI models to different data sources and tools in the pet boarding business domain.

Available Testing Tools

Test the Server-Sent Events (SSE) implementation of MCP with a browser-based client.

This tool allows you to see real-time context streaming in action.

API Documentation

Context Endpoints

MCP provides both REST and SSE endpoints for retrieving context:

GET /api/mcp/context - REST API endpoint
GET /api/mcp/context-stream - SSE streaming endpoint

Action Endpoints

Perform actions through the MCP:

POST /api/mcp/actions - Execute actions like booking appointments

Integration with n8n

The MCP implementation is designed to work seamlessly with n8n workflows, allowing for AI-powered automations.

Example use cases include:

  • AI-powered chatbots and voice agents
  • Automated availability search
  • Intelligent booking assistants
  • Notification services (SMS, email)
